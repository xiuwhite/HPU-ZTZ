程序环境说明：采用Python 3.10 环境编写，IDE采用Pycharm环境
该程序需要多线程环境，请寻找能同时执行多个py文件的IDE(例如Pycharm)

————————————————————————
环境配置：
该项目文件包中的requirements.txt文件为项目的依赖包，需要先安装下载这些包到你的解释器才能进行运行
你需要在cmd窗口里切换到该项目文件目录下，在确保您的电脑拥有python3.x以上环境的情况下，输入命令
安装命令为：pip install -r requirements.txt

————————————————————————————————
程序执行：
该项目需要先运行server.py文件，使得服务端跑起来，并且不能关
然后运行win_login.py文件，登陆到FTP服务端，然后弹出一个客户端功能界面，选择你要执行的操作即可
